# URL Shortener with Login & Analytics

A full-stack URL shortener built with **Node.js, Express, MongoDB, and EJS**.  
Users can **signup/login**, generate **short URLs**, and track **click analytics**.

## Features

- User signup and login
- Generate short URLs
- Redirect to original URLs
- Track clicks and visit history
- View all URLs created by the user
- Protected routes (only accessible to logged-in users)
- Beautiful EJS front-end with a starry theme

## Tech Stack

- **Backend:** Node.js, Express
- **Database:** MongoDB (Mongoose)
- **Frontend:** EJS, HTML, CSS
- **Authentication:** JWT tokens in cookies
